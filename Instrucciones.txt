Instrucciones:
1. Base de Datos
- Ingresar al PostgreSQL y crear una bd con los siguientes datos:
Database: pruebaTecnicaDB
Owner: postgres
- Importar los datos que se encuentran en el archivo prueba pruebaTecnicaDB.sql

->Luego de clonar el proyecto:
2. Backend
Con el terminal nos posicionamos en la carpeta PruebaTecnica y digitamos el comando: 
npm install
npm run dev / npm start 

3. Frontend
 Abrimos otro terminal, nos posicionamos en la carpeta PruebaTecnica/client y digitamos el comando:
 npm install 
 npm run dev / npm start
